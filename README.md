# Full-Stack Web Development Course (0 → Pro)

A beginner-to-pro, hands-on course covering the fundamentals of modern web development.

Contents
- Day 1/: Intro to HTML — includes a simple `index.html` (Hello World).

Getting started
- Open the Day 1 example in your browser:

	macOS:
	```bash
	open "Day 1/index.html"
	```

Course structure
- Day folders: each day contains example files and exercises.
